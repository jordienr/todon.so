Free Mastodon.social Link Shortener

This is a free link shortener for Mastodon.social. It is not affiliated with Mastodon.social in any way.

How to use it

1. Go to `{mastodon_username}.todon.so`

2. You'll be redirected to mastodon.social/@{mastodon_username}

3. You can now use this link anywhere

Example

jordienr.todon.so will redirect you to https://mastodon.social/@jordienr
